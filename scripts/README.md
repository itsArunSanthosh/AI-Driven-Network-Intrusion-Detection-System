# Scripts

Utility scripts to run different parts of the system.

## Available Scripts

- run_ingestion.sh → Starts traffic generation
- run_streaming.sh → Starts streaming pipeline
- run_training.sh → Runs training pipeline
- run_inference.sh → Starts API server
- run_all.sh → Runs full system
- setup_env.sh → Sets up Python environment

## Usage

```bash
bash scripts/run_all.sh
